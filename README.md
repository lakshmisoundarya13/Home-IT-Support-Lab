# 🖥️ Home IT Support Lab — Simulated Enterprise Environment

## Project Overview
This project demonstrates a fully functional simulated enterprise IT environment built using VirtualBox. It covers the core skills required for IT Support, Helpdesk, and System Administration roles including Active Directory, Group Policy, Domain Networking, and Security Monitoring.

---

## 🏗️ Lab Architecture

| Machine | OS | Role | IP Address |
|---|---|---|---|
| DC01 | Windows Server 2019 | Domain Controller | 192.168.1.1 |
| CLIENT01 | Windows 11 | Domain Client | 192.168.1.2 |

---

## ✅ Skills Demonstrated

- Windows Server 2019 installation and configuration
- Active Directory Domain Services (AD DS) setup
- DNS Server configuration
- Domain creation (lab.local)
- Organisational Unit (OU) management
- User account creation and management
- Security Group configuration
- Group Policy Object (GPO) configuration
- Password Policy enforcement
- Account Lockout Policy
- Static IP configuration and network connectivity
- Joining Windows 11 client to Active Directory domain
- Domain user authentication
- Security event monitoring using Event Viewer (Event ID 4624)
